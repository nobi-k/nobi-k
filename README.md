### Маркетолог и аналитик, который вайбкодит свои продукты

Сначала — маркетинг и аналитика: гипотезы, цифры, воронки, рост.<br>
Потом — вайбкодинг: довожу идею до продукта в проде сам, без команды разработки.<br>
Маркетинг + аналитика + AI = работающие сервисы, а не презентации.
